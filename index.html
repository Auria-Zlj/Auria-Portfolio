    Design Response / Solution

    We designed a connected system combining dual wearable sensors, a guidance app with real-time feedback, and a clinician dashboard. This closes the loop between home exercises and professional oversight, restoring confidence and accuracy in recovery.

    Design Strategy

    We designed a dual-sided UX system addressing both patient and clinician needs:

    1. Patient App ‚Äì real-time motion feedback, progress tracking, and guidance.
    2. Clinician Dashboard ‚Äì remote monitoring and automated reporting.
    3. Data Pipeline ‚Äì BLE sensors stream angle and distance data to Firebase and auto-generate reports.

    Each component forms a continuous feedback loop between human movement and data insight.

    Experience Flow

    1. Patient starts a guided exercise ‚Üí receives real-time visual feedback.
    2. System detects hold duration ‚Üí counts repetitions accurately.
    3. Session data uploads to the cloud ‚Üí clinician dashboard updates instantly.
    4. Progress report auto-generates ‚Üí therapist adjusts rehab plan remotely.
    Result: A continuous experience connecting human motion, interface, and medical supervision.

    UX Design Highlights

    • Real-Time Feedback: Color-coded visuals and countdowns built trust and reduced uncertainty.
    • Progress Visualization: Graphs reinforced motivation during long recovery cycles.
    • Error Prevention: Range-of-motion cues prevented unsafe movements.
    • Report Automation: Simplified data visualization for clinician clarity.

    Hardware Integration

    Two BLE modules capture knee angle and lift distance with adaptive orientation design. The UX design incorporated hardware latency and calibration constraints to ensure smooth, accurate feedback.

    Outcome & Impact

    • Increased patient confidence and consistency.
    • Enabled remote supervision and data-driven insights for clinicians.
    • Reduced motion feedback latency from minutes to milliseconds.
    • Prototype successfully integrated into the CareLink+ 5G test environment.
